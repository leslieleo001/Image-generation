Silicon Flow AI图片生成助手 - 使用说明
================================

1. 运行环境要求
---------------
- Windows 操作系统
- 不需要安装 Python 或其他依赖
- 建议安装最新版本的 Visual C++ Redistributable（如果运行时提示缺少DLL）
  下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe

2. 安装步骤
---------------
1) 创建一个新文件夹（例如：D:\Silicon Flow）
2) 将 Silicon Flow.exe 复制到这个文件夹中
3) 双击运行程序，它会自动创建必要的文件夹：
   - config：配置文件
   - output：生成的图片
   - presets：预设配置
   - history：历史记录

3. 使用注意事项
---------------
- 请不要将程序放在需要管理员权限的目录（如 Program Files）
- 确保程序所在目录有写入权限
- 首次运行时可能会被杀毒软件拦截，请允许程序运行
- 建议将输出文件夹添加到杀毒软件的白名单中

4. 基本使用说明
---------------
1) 手动生成
   - 在提示词输入框中输入描述文字
   - 选择合适的模型和参数
   - 点击"生成"按钮

2) 批量生成
   - 切换到"批量生成"标签页
   - 导入或编辑提示词列表
   - 设置生成参数
   - 点击"开始生成"按钮

3) 历史记录
   - 点击"历史记录"按钮查看之前生成的图片
   - 可以复制使用过的提示词
   - 可以删除不需要的记录

5. 常见问题解决
---------------
Q: 程序无法启动，提示缺少DLL？
A: 请安装 Visual C++ Redistributable 2022

Q: 杀毒软件报警？
A: 这是正常现象，因为程序是打包的exe。请将程序添加到杀毒软件的白名单中。

Q: 生成的图片在哪里？
A: 在程序所在目录的 output 文件夹中。

Q: 如何备份我的设置和历史记录？
A: 备份整个程序目录即可，主要是 config 和 history 文件夹。
